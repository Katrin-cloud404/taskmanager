# TaskMaster 📝
A simple and lightweight **Command Line Task Manager** written in Python.  
It lets you add, delete, view, and mark tasks as completed — all stored locally in a JSON file.

---

## 🚀 Features
- Add new tasks  
- Mark tasks as complete  
- Delete tasks  
- View all tasks  
- Persistent storage using `tasks.json`  

---

## 📦 Installation
git clone https://github.com/katrin-cloud404/taskmaster.git
